#  College Majors Analytics: Salary, Employment & Gender
#  Kaggle Notbook
 https://www.kaggle.com/code/talalhakem/collegemajors-analytics-salary-employment-gender
##  Project Overview
This project presents an Exploratory Data Analysis (EDA) of the College Majors dataset to uncover insights into salary, employment, unemployment, and gender distribution across different academic majors.
The analysis uses Python and data visualization techniques to identify trends and relationships within the dataset.
##  Objectives
- Explore the dataset
- Clean and prepare the data
- Analyze salary distribution
- Examine employment and unemployment rates
- Compare gender representation across majors
- Visualize key insights
## Dataset
The dataset includes information about:
- College Major
- Major Category
- Median Salary
- Employment
- Unemployment
- Unemployment Rate
- Full-Time Employment
- Share of Women
- Number of Men & Women
